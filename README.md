Задание 4

Подключить созданное хранилище Redux к вашему приложению React с использованием компонента `Provider` из библиотеки `react-redux`.

1. Откройте файл index.js:

Найдите файл `index.js` в корневой директории вашего проекта.

2. Импортируйте компонент Provider и хранилище:

Импортируйте компонент `Provider` из библиотеки `react-redux`.
Импортируйте хранилище из файла `store.js`, которое вы создали в предыдущем задании.

3. Обертка компонента App в Provider:

Найдите место, где ваш компонент `App` рендерится в DOM.
Оберните компонент `App` в компонент `Provider`.
Передайте хранилище в качестве свойства `store` компонента `Provider`.

4. Запустите приложение:

Запустите ваше приложение и убедитесь, что оно работает корректно.
